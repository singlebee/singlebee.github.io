2024#14 ｜ 清明加班是在给自己烧香

2024/04/06 4464 个字
阅读本文将浪费: 9 分钟生命

依靠PicList和Image Auto Upload，加图片特别轻松！疯狂插图片进来哈哈哈！不过清明三天假期就咸了一天，剩下的日子都没休息到。

LocalSend

iphone想把图片导入到PC上挺麻烦的，官方的说法需要ituns，那玩意老难用了，直接插数据线好像也读取不到文件。我三星手机发文件给pc就挺简单的，直接将图片往三星多屏联动上一拖就可以了。

在网上搜了搜，搜到了LocalSend这个软件，理解成第三方的三星flow就可以了，用于苹果和pc传文件的体验还是不错的。

原理应该和三星flow是一样的，只要把两个设备连上同一个网络，就能通过局域网传文件了，传文件速度很快，解决了pc和ios之间传文件麻烦的痛点。

将图床改为了COS

之前一直在用免费图床vgy，这个免费图床是在上传工具shearX上看到的，国内的连接速度其实还可以，目前也没出什么问题。

唯一的困扰就是每次写博客要插入图片好麻烦，需要先把图片压制一遍，然后上传到免费图床上面，ShearX的好处之一就是能直接把图片的markdown格式直接复制过来。但是还是好麻烦。对免费图床安全性没把握，加上冲浪的时候看到了腾讯的COS似乎挺便宜的？

在嘟友sky酱的帮助下，我把图床全部迁移到了COS。

本来以为会蛮麻烦的，一开始想的是要写个python脚本，把原来md文件里的图片全爬下来，然后再传到COS上，同时再替换掉原来的链接，挺困扰的。

但是！sky酱推荐的PicList社区里已经有人做过这个功能的插件了！喜闻乐见大快人心普天同庆奔走相告！

把整个post文件夹用这个插件打开，它就会开始遍历所有md文件里的图片链接，然后自动上传到你图床上，并替换原来的链接！

啊啊啊，太方便了！直接把图片粘贴到Obsidian里来，在它们两个协同下会自动压制成webp格式，然后上传到图床上，啊啊啊，太方便了。

边上班边写还是有点迷糊，写过的东西总重复来写。

效果如上，大家基本上有这个功能的都是抄自 Hi, NeoDB这篇，我基于这篇稍微改了改样式，凑合用着了。

因为年初的时候正好用为了整理一份豆瓣导出的数据，就用到了NeoDB的api，所以这个他这个代码大概分析下就很清楚还有很多玩法。

比如我的代码里多引入了$dbFetch.orig_title、$dbFetch.genre 、$dbFetch.area分别对应原始标题、类型和地区，在返回的json数据里都有的，直接可以调用。

因为有些东西可能会返回数组类型，有些需要改动一下。

<div class="db-card-area">
	{{ with $dbFetch.area }}
	{{ range $index, $area := . }}
	{{ $area }}
	{{ end }}
	{{ end }}
</div>

虽然不知道这种写法对不对，反正能用了，能用了就不管了。

如龙8

人中之龙8
BBQ评分：
角色扮演 冒险 策略 动作
从“人生低谷”东山再起的男人——春日一番， 以及面临“人生最后”之战的男人——桐生一马。 由两位主角演绎的超大作RPG， 系列至高的感人故事。 战斗进化为更具策略性的“动态指令RPG战斗”。 通过新增的职业，丰富的自定义玩法及养成要素，创造最强的队伍吧！ 故事的舞台发生在异国他乡的“夏威夷”，为系列首度。 玩转各色夏威夷独特的娱乐场所吧！
game
目前还没通关，玩法承接上一代如龙7，从以前的动作类游戏变成了rpg游戏，主角从硬汉风的桐生一马变成了谐星风的春日一番。如龙8这一代的地图从神室町和异人町扩展到了夏威夷，两个地区都有各自的剧情和玩法。

如龙的回合制和别的回合制不太一样的是，战斗时还可以考虑时间与空间，aoe释放的角度和时机会影响aoe的效果，以及队友的站位会影响是否能合体攻击、补刀等等。平常最喜欢的就是触发战斗的时候立马用一个圆圈aoe，因为一般刚开始的时候敌人都是抱团。

友情技我不怎么用，因为用过几次感觉还不如桐生哥一个过肩摔伤害高。

剧情其实还是和以前一样，有些地方挺随便的。

春日因为给黑道介绍工作被开除了（多多良造谣的什么来着忘了），难波因为和公司秘书关系好就被开除了，阿英居然仅仅是因为一个车祸就被开除了，日本人如履薄冰的一生…..

桐生哥回日本后，会解锁双主角双线叙事，桐生哥会解锁一个「死前日记」的系统，时不时回忆一下以前的剧情。我只玩过如龙0，很多剧情毫无代入感，只是感慨桐生哥你人生好精彩！

其实双线叙事很烦的就是练级，桐生哥回日本之后，有个剧情是叫我去练级，我打完之后好像是38级，花开两朵各表一枝回到夏威夷之后，春日32级，我焦虑不知道能不能去推主线于是硬着头皮又刷到36级，刷累了去推主线发现主线里敌人等级也就32……

游戏体验上感觉在玩gta，还有自己的宝可梦和无人岛玩法…….玩法太多……

以及游戏里春日已经原谅了不二宫（多多良）但我还没有！但是不二宫是我的强力C，我又不能没有她！

嗯，还没通关，但是我挺推荐玩的！当然最好是先玩如龙7，有兴趣的还能玩如龙0和番外死神之眼。喜欢木村拓哉嘛？喜欢就来玩死神之眼吧！打开游戏你就是木村拓哉！（这三个我都没通关）
